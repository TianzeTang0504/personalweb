# Fonts

`writer.html` uses Alibaba PuHuiTi 3.0 for the writing workspace UI.

Included files:
- `AlibabaPuHuiTi-3-55-Regular.woff2`
- `AlibabaPuHuiTi-3-65-Medium.woff2`

The official Alibaba font download endpoint blocked direct access in this environment, so these two webfont files were downloaded from the jsDelivr npm mirror packages for the matching PuHuiTi 3.0 weights. If replacing them later, prefer Alibaba's official font site/download channel.
